MM PARK

About

First Program I ever made! An Information System that allows the management of parking lots. Keep track of every user entrance and other features.

Made mainly to learn Object Oriented Programming and some basics of Graphic Design, the platform allows users to create a park and to record the inputs and outputs of the same and users of the system so that users who do not use the park can view and manage the program.
