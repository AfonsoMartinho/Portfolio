RUNNING PUMPKIN

About

Running Pumpkin is a browser 2D platform game where a pumpkin runs to his home castle! Will you be able to help the still homeless pumpkin?

To help the poor living being the player needs to control the rushed pumpkin with is keyboard avoiding ground and air obstacles such as golden blocks and rockets. The character will count with the help of grass platforms that he can jump in.

The Game

The character runs automatically, which means that that player only needs to press jump keyboard button (W Key) to climb the grass platforms and dodge the multiple obstacles encountered.

The more the pumpkin runs the more points the user gets. When the pumpkin collides with an obstacle or reaches is home castle the game ends. When that happens, the user needs to do a math calculation to gain some extra points and maybe reach is high score!

Tools

The project was created with the HTML canvas element using no game engine whatsoever.
